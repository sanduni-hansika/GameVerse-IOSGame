# 🎮 GameVerse – Multi-Game iOS App

GameVerse is a modern, interactive iOS gaming app built with SwiftUI that brings together multiple mini-games in one place. It offers fast-paced gameplay, score tracking, leaderboards, location-based game history, and an intuitive user experience designed for casual players.

 ✨ Features

- 🎯 **Three Mini Games**
  - Tap Frenzy
  - Light It Up
  - Quiz Rush

- 👤 **Player Profiles**
  - Enter your player name before each game
  - Personalized gameplay experience

- 🏆 **Leaderboard System**
  - Stores top scores for every game
  - Highlights new personal bests

- 📊 **Statistics Dashboard**
  - View recent game sessions
  - Track scores and performance history

- 🗺️ **Game Location Services**
  - Records game session locations using device location services
  - Automatically captures the player's location when a game is completed

- 🎮 **Dynamic Game Difficulty**
  - Progressive levels and increasing challenge
  - Interactive animations and visual effects

- ⏱️ **Real-Time Gameplay**
  - Countdown timers
  - Live score updates
  - Instant game feedback

- 📱 **Modern SwiftUI Interface**
  - Clean and responsive design
  - Smooth animations
  - Dark-themed user interface

- 💾 **Local Data Storage**
  - Saves scores and game history locally
  - Persists leaderboard data between app launches

---

## 🕹️ Included Games

### 🔴 Tap Frenzy
Tap the moving button as many times as possible before the timer runs out.

### 💡 Light It Up
Tap the blooming flower tiles before it wilts. Watch smooth flower bloom and fade animations while advancing through faster, more challenging levels.

### 🧠 Quiz Rush
Answer multiple-choice trivia questions, build streaks, and earn bonus points.
